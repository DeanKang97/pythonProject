# 진짜 리뷰 판별 사이트 제작 프로젝트

"소정의 원고료" 라는 키워드가 들어간 광고성 블로그 게시글을 웹 크롤링(Python)한 후,</br>
 AI학습을 통해 광고성 게시글과 아닌 게시글의 언어적 특성(명사, 부사, 형용사 등)을 분석, 분류하여 </br>
 소비자에게 신뢰도 있는 리뷰를 제공,합리적 소비를 하게끔 도움을 주는 사이트입니다. </br></br>
